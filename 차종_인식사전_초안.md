# 차종 인식 사전 — ★초안 (시경 교정용)

> **용도**: Phase C 차종 스캐너가 수확한 키워드에서 "진짜 차종"을 걸러내고 정규명으로 통일하는 **인식·정규화 사전**. (검색 시드용 아님.)
> **읽는 법**: `정규명 | 별칭(사람들이 검색하는 형태들, 쉼표구분) | 비고`. 별칭이 키워드에서 발견되면 정규명 행으로 합산.
> **★시경 교정 포인트 (이 사전의 권위는 시경께 있습니다 — 부품을 직접 파니 세대·표기를 저보다 잘 압니다)**:
> 1. **세대코드·페이스리프트 명칭 매핑이 가장 불확실합니다.** 아래 "(매핑 검증)" 표시된 곳, 특히 "올뉴/더뉴/신형"이 어느 세대에 붙는지 — 상세페이지 표기 기준으로 바로잡아 주세요.
> 2. **도로에 거의 없는 세대는 빼도 됩니다** (부품 수요 미미). 반대로 빠진 모델·별칭 추가.
> 3. **수입차는 시작 세트만**입니다 — 국내 부품 수요 큰 모델 위주로 보강.
> 4. C-2 실험이 "버려진 모델처럼 보이는 토큰"을 출력하니, 그걸 보고 누락분을 채우면 됩니다.

## 정규화 주의사항 (Claude Code 참고)
- bare 모델명(세대 없이 "아반떼", "쏘나타")은 모호 → **별도 버킷**으로 보존(임의로 특정 세대에 귀속 금지). 사람이 해석.
- 별칭 매칭은 공백·대소문자 정규화 후. "쏘렌토 MQ4" = "쏘렌토MQ4".
- 한 키워드에 여러 모델명이 없도록 가정하되, 충돌 시 더 구체적인(세대 포함) 쪽 우선.
