# dgk-product-scout

신제품 발굴 자동화 시스템 (2-레이어). 본 리포는 **P1 — Layer 1 카테고리 발굴 엔진**만 구현한 상태다.

> Layer 1 = "어떤 제품군에 진입할까"를 찾아 **카테고리 후보 랭킹**을 출력한다.
> Layer 2(SKU 세부 검증)는 후속 Phase.

---

## 빠른 시작

```bash
pip install -r requirements.txt

# 카테고리 랭킹 대시보드 (현재 MockAdapter 기반)
streamlit run app.py

# 테스트
pytest
```

`streamlit run app.py` 를 실행하면 mock 데이터 기반 카테고리 랭킹이 화면에 뜬다.

## 구조

```
dgk-product-scout/
├─ config.py                 # 모든 임계값·요율·경계·가중치 (하드코딩 금지, 설명 주석)
├─ app.py                    # Streamlit 랭킹 대시보드
├─ src/
│  ├─ schema.py              # 3.4 category_candidate + 원시 관측치 스키마
│  ├─ signals.py             # 3.2 발굴 신호 8개 산출
│  ├─ winnability.py         # 3.3 진입 가능성 4분면 판정
│  ├─ ranking.py             # 3.4 후보 생성 + 랭킹 + 함정 필터
│  └─ adapters/
│     ├─ base.py             # DataAdapter 인터페이스 (수집부 계약만)
│     └─ mock_adapter.py     # stub + mock 데이터 (가짜 값 명시)
└─ tests/                    # pytest (신호·4분면·랭킹)
```

## 구현 범위 (P1, 스펙 3절)

- **카테고리 후보 레코드 스키마(3.4)** — `src/schema.py`.
- **발굴 신호 8개(3.2)** — `src/signals.py`. 각 신호 0.0~1.0 정규화.
- **진입 가능성 4분면 판정(3.3)** — `src/winnability.py` (최우선/틈새/함정/무시).
- **카테고리 랭킹(3.4)** — `src/ranking.py`. 큰 시장 × 가치 싸움 우선.

### 함정(광고 싸움) 처리 정책

4분면의 **'함정'(큰 시장 × 광고 싸움)** 은 점수 감점이 아니라 **기본 제외(필터)** 다.
큰 시장이라도 광고 의존도가 높으면 랭킹에서 빼서, 감점을 뚫고 상위로 오지 못하게 한다.
단, 대시보드 사이드바의 **`함정 포함 보기`** 토글로 다시 포함해 볼 수 있다.
(정책 값은 `config.EXCLUDE_TRAP_BY_DEFAULT` 등으로 조정.)

## ⚠️ 미확정 항목 — stub 교체 안내

본 P1 구현은 **외부 데이터 수집부를 stub(MockAdapter) + mock 데이터로 채웠다.**
임의의 스크래핑 방식이나 채널 수수료 값을 지어내지 않았다.

- **스펙 12절 #1 (소스별 실제 수집 방식)** — 쿠팡·1688 등을 공식 API / 자동 스크래핑 / 수동 중
  무엇으로 수집할지 미확정.
- **스펙 12절 #2 (채널 수수료 요율 값)** — 플랫폼×카테고리 테이블에 채울 실제 요율 미확정.

> **12절 #1·#2 확정 후 stub(`src/adapters/mock_adapter.py`)을 실제 수집부로 교체**한다.
> 교체 시 `src/adapters/base.py` 의 `DataAdapter` 인터페이스를 구현하는 실제 어댑터
> (예: 쿠팡 API 어댑터, 1688 어댑터)를 만들어 `app.py`/`discover()` 에 주입하면 된다.
> 신호 산출·4분면·랭킹 로직은 그대로 재사용된다.

  기반 비소모품 기회 포착) — **P1 범위 밖이며 후속 Phase로 보류.**
  현재 엔진은 주 엔진인 "내구재 + 정기 교체 소모품" 패턴 중심으로 동작한다.
- **Layer 2 (SKU 세부 검증 6단계)** — P2 이후.
- 관세율(③)·고정단가(⑥) 정밀 마진 — 스펙 8절 보류, 실제 진입 직전 입력.
